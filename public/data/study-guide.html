概念 定义 关键词
划分（Allocation） 将特定频段列入频率划分表，规定可供某业务使用 频段 + 业务

⚠️ 重点 考试陷阱

三个词极易混淆。记忆口诀：划频段给业务、分频道给部门、指频率给电台。

监督与处罚

  • 依法对业余电台实施监督管理的机构：国家无线电管理机构和省、自治区、直辖市无线电管理机构（不是业余无线电协会，不是体育管理机构，不是公安部门）
  • 业余业务是指：供业余无线电爱好者进行自我训练、相互通信和技术研究的无线电通信业务
  • 卫星业余业务：利用地球卫星上的空间电台开展与业余业务相同目的的业务

常见考试陷阱

  • 业余无线电协会不具有管理执法权，不能核发执照或呼号
  • 体育管理机构与业余无线电管理无关
  • 业余电台只能在业余业务和卫星业余业务范围内通信，不能与其他业务电台通信（紧急情况除外）
  • 未经批准，业余电台不得进行广播或发射通播性质的信号

  • 无线电管理机构颁发业余电台执照时，应当同时向申请人核发业余电台呼号
  • 申请人取得呼号后，再次申请设置、使用业余电台，无线电管理机构不再核发新的呼号
  • 注销的呼号重新投入分配前，申请人再次申请时，应核发已注销的原呼号
  • 对已核发的呼号不满意，不可以申请另行核发
⚠️ 重点 一人一呼号原则

每个业余无线电爱好者终身只有一个呼号。换设备、换地址都不换呼号。

呼号前缀与分区

  • 呼号前缀由国家无线电管理机构编制和分配
  • 必须使用完整呼号，任何时候不能用后缀替代完整呼号（即使与友台联络也不例外）
  • 地区码加后缀不等于完整呼号，不具标识意义

⚠️ 重点 考试陷阱

申请条件

  • 年龄条件：未成年人也可以申请设置、使用规定条件内的业余无线电台（不限制最低年龄，但需要具备相应能力）
  • 不是业余无线电爱好者而申请设台（如仅为行车方便）：错误，属于不具备"熟悉无线电管理规定"设台条件的表现
  • 发射设备要求：可以使用自制、改装、拼装等未取得型号核准的设备（无线电管理机构进行技术检测时不得收取任何费用）

申请流程

台址类型 受理机构
有固定台址 电台所在地的省、自治区、直辖市无线电管理机构
没有固定台址 申请人住所地的省、自治区、直辖市无线电管理机构

执照管理

  • 执照有效期不超过：5 年
  • 有效期届满后继续使用：应在期限届满30 个工作日前申请更换
  • 变更执照载明事项：应及时向作出许可决定的无线电管理机构申请办理变更手续
  • 终止使用：向作出许可决定的无线电管理机构办理注销
  • 多台发射设备：视为一个业余电台，指配一个呼号，但所有设备均应经过核定并载入电台执照

合法设台的必要步骤

按照《业余无线电台管理办法》向无线电管理机构提出申请，取得业余无线电台执照。不需要加入协会、不需要单位批准。

⚠️ 重点 关键数字

- 执照有效期：5 年

- 续期申请提前：30 个工作日

- 技术检测：免费

1.4 操作权限（127题）

A、B、C 三类操作证权限

类别 频率范围 最大发射功率
A 类 30-3000MHz 频段 不大于 25 瓦
B 类 30MHz 以下频段小于 15 瓦，或 30MHz 以上频段不大于 25 瓦 见频率范围
C 类 30MHz 以下频段不大于 1000 瓦，或 30MHz 以上频段不大于 25 瓦 见频率范围
⚠️ 重点 必记

- A类：只能 VHF/UHF，最高 25W

- B类：30MHz 以下 < 15W，30MHz 以上 ≤ 25W

- C类：30MHz 以下 ≤ 1000W，30MHz 以上 ≤ 25W

- 特别关注：A类不能上短波，C类在 30MHz 以下可达 1000W

验证与考试机构

  • 可以组织 A 类和 B 类业余电台操作技术能力验证的机构：国家无线电管理机构和省、自治区、直辖市无线电管理机构
  • C 类操作技术能力验证：由国家无线电管理机构组织

频率使用的基本原则

  • 业余电台在核准使用的频段内，享有平等的频率使用权（不同类别业余电台之间没有优先级差别）
  • 作为次要业务时：不得对主要业务产生有害干扰；不得对主要业务的干扰提出保护要求；可要求同一业务或其他次要业务不对自己产生干扰

发射类别

  • 用标准符号标示发射的一组特性，包括主载波调制方式、调制信号、被发送信息类型等

VHF/UHF 主要业务频率

  • VHF 和 UHF 范围内业余业务作为主要业务的频段：50MHz、144MHz
  • 我国业余频率最高频段：248-250GHz，唯一主要业务
⚠️ 重点 考试陷阱

- 430MHz（70厘米波段）在我国是次要业务，不是主要业务！这是高频考点。

- 144-146MHz 是唯一主要业务，146-148MHz 是与其他业务共同作为主要业务

- 业余专用频段（30MHz以下）：7MHz、14MHz、21MHz、28MHz、47GHz

WARC 频段（1979年世界无线电行政大会增配）

WARC 频段 = 三个 HF 业余频段：10.1-10.15MHz、18.068-18.168MHz、24.89-24.99MHz

第二章 操作规范

2.1 通联程序（86题）

呼叫规范

  • CQ 的含义：非特指地呼叫任何国内和国外业余电台
  • 发起 CQ 呼叫的正确方法：呼叫 "CQ" 并报出己方呼号，可重复一两次 CQ 并对呼号进行字母拼读
  • 发起呼叫前应先守听一段时间，然后用 "QRL?" 询问频率是否空闲
  • 不能用 "Calling you, Roger?" 来表达"有人使用频率吗？"

频率使用注意

频段 本地联络应避免占用的频率
144MHz 144-144.035MHz 和 145.8-146MHz
430MHz 431.9-432.240MHz 和 435-438MHz

通联行为规范

  • 禁止转播广播电台、互联网聊天、电话通话内容（错误行为，违反"只能用于相互通信、技术研究和自我训练"）
  • 禁止为他人转达信息（违法行为，不是"无偿就合法"）
  • 禁止用强信号压制其他电台（违法行为，违反严禁阻碍其他无线电台通信的规定）
  • 遇到非业余电台干扰：不应在频率上回应，应收集信息并向无线电管理机构举报
  • 发现其他业余电台操作不规范但不严重：通过电话、邮件等方式私下善意建议
  • 未经批准不得进行广播或发射通播性信号（即使是向会员播发技术讲座也违法）

通联语言

  • 任何时候都应使用明语或业余无线电公认的缩略语、简语
  • 不能使用自创的缩略语或暗语
  • 不能使用只有部分人能懂的稀有语言或方言来排斥其他电台

通信方式选择

  • 与几十千米以外有很多重要问题要交流：应使用电话或互联网（业余电台应尽量简洁，复杂事务不要占用业余频率）

常用频率约定

  • 10.1MHz 以下频段使用下边带（LSB），10.1MHz 以上频段使用上边带（USB）——这是各国爱好者长期形成的通联习惯，非技术原因

比赛规则

  • CQWW 比赛中，SOAB（单人全波段）组别不能使用 CW Skimmer 辅助发现 QSO 机会

中继台使用

  • 除必要的简短联络外，中继台频率应保持空闲，以备紧急情况响应
  • 中继台向其覆盖区域内所有业余电台提供平等服务，技术参数应公开
  • 不能因出资建设而拒绝其他业余电台使用
  • 中继台收费使用是禁止的

梅登海德网格定位

  • 4 字符网格：经度 2 度 × 纬度 1 度
  • 6 字符网格：经度 5 分 × 纬度 2.5 分

天线方向参考

  • 联络欧洲时天线指约 300 度（十点钟），欧洲联络我们时指约 60 度（两点钟）——相差 180 度

紧急通信

  • 业余电台允许与非业余电台通信的唯一条件：突发重大自然灾害等紧急情况，内容限于与抢险救灾直接相关的紧急事务

FM 频率调制（调频） Frequency Modulation

CW 与数字模式

  • CW = 等幅电报（Continuous Wave），不等同于莫尔斯编码
  • RTTY = 频移电报技术（frequency-shift telegraphy），电报信号控制载波频率在预定范围内变化
  • 单边带发射：只传送一个边带的调幅发射（不含载波）

发射的定义与类别

  • 发射（emission）：由无线电发信电台产生的辐射或辐射产物
  • 业余电台的杂散产物是一种发射
  • 闪电产生的电磁波不是发射（不是电台产生的）
  • 接收机本地振荡器的辐射不是发射
  • 医用高频电疗机的能量不是发射类型中的"发射"
  • 带外发射：由于调制过程而产生的、刚超出必要带宽的一个或多个频率的发射（杂散发射除外）

数据传输概念

  • 符号速率：描述信号状态变化的快慢（单位：波特）
  • 数据速率：描述信息流量的快慢（单位：比特/秒）
  • MFSK16 符号速率低于数据速率（每个符号携带多比特），抗突发干扰能力更强
  • RTTY 符号速率等于数据速率（每个符号只携带 1 比特）

数字模式选频

  • FT8 通联时应：在对方守听时的瀑布时段中找相对清净的频率
  • DRM 数字慢扫描电视（DSSTV）：倾向于通过单边带传输，保留基带特性，利于衰落信道下的差错控制

PASS（PACKET通信）

  • 业余无线电中使用 AX.25 协议的数据分组通信方式

卫星通信功率控制

  • 应确保下行链路信号强度与信标大体一致
  • 过强的上行信号会压低转发器对其他信道的转发功率

广播的禁止

  • 业余电台不得以任何方式进行广播或发射通播性信号（即使内容有益也不允许）
  • 遇非业余电台干扰时，不应在频率上向对方宣传法规（违反"只能用于相互通信和技术研究"的规定）

3.3 接收与信号处理（44题）

AGC（自动增益控制）

  • 功能：对中频级信号电平进行检测并反馈控制，防止电路过载
  • 关闭 AGC 的情况：有用微弱信号和强干扰同时出现时
  • 选择较短 AGC 时间常数：接收 FM/FSK/PSK 等包络幅度恒定的信号
  • 选择较长 AGC 时间常数：接收 SSB 和 AM 等包络幅度不断变化的信号
⚠️ 重点 AGC 时间常数记忆口诀

- 短 AGC = 恒定包络（FM/FSK/PSK）

- 长 AGC = 变化包络（SSB/AM）

ATT（衰减器）

  • 作用：特强带外信号可阻塞接收机前级电路，造成失真和互调，需在接收机前端加入衰减器
  • ATT 不是为了扩展增益控制范围，也不是为了防止本台发射机强信号损坏接收机

其他面板功能

符号 功能

无意接收非业余业务信息的处理

  • 不得传播、公布（即使是与其他爱好者私下交流也不允许）

卫星信号的快衰落

  • 原因：卫星翻滚造成收发信天线极化的失配（不是电离层影响）

强噪音环境下接收微弱信号

  • 合理的做法：尽量只呼叫和回答能听到的电台，发起 CQ 呼叫时应降低功率（不是增大功率来压倒噪声）

  • 主要用途：RTTY
  • 编码属性：每个码包含 5 个二进制数据，可以换态扩位

接收机架构

  • 超外差式：射频信号 → 混频 → 中频（IF）→ 解调。混频器将信号频率变换为另一个频率
  • 上变频：中频频率 > 输入频率
  • 下变频：中频频率 < 输入频率
  • DC 式（直接变换，Direct-Conversion）：射频信号在解调前不做频率变换，直接解调
  • SDR（软件无线电）：主要功能由 DSP 算法实现，无需物理滤波元件即可现场调节滤波特性

噪声与灵敏度

  • 信噪比（SNR） = 有用信号功率 / 噪声功率
  • 噪声系数 Fn = (Si/Ni) / (So/No) — 输入端信噪比与输出端信噪比的比值
  • Fn > 1（任何实际接收机）；越接近 1 越好
  • 理想无噪声：Fn = 1
  • 噪声指数 NF = 10 log(Fn) — Fn 的对数形式
  • NF > 0dB；越接近 0 越好
  • 理想无噪声：NF = 0dB
  • 噪声温度 Te：接收机内部噪声等效为一个匹配电阻在绝对温度 Te 下产生的热噪声
  • 理想无噪声：Te = 0°K
  • 天线放大器在 17°C 下，内部噪声等效输入热噪声 → Fn=2, NF=3dB, Te=290°K
  • 制约现代接收机灵敏度的主要因素：机内噪声（不是外部干扰）
  • 静噪灵敏度：能使静噪电路退出静噪状态的最小射频输入电平

S 表（信号强度指示）

  • S1 至 S9，每档间隔 6dB；S9 以上为 +10dB、+20dB 等
  • S 表读数下降 N 档：功率变化 = 10^((N×6)/10) 倍
  • 功率从 100W 降到 25W（1/4，-6dB）：S8 → S7
  • 功率从 15W 到 S4 时：S9→S4 = 5档 = 30dB → 功率 = 15W ÷ 1000 ≈ 0.015W
  • 功率从 100W 到 S5 时：S8→S5 = 3档 = 18dB → 功率 = 100W ÷ 63 ≈ 1.6W（实际为 100/64≈1.56W）

DSP 与 SDR

  • ADC 采样率：必须 ≥ 输入信号最高频率分量的 2 倍（奈奎斯特定理）
  • ADC 量化精度：取决于输入信号的波形复杂程度
  • I/Q 信号混频：使用复数乘法算法
  • FFT（快速傅里叶变换）：用于频谱分析、信号解调等多方面
  • FPGA：用于高速并行数字信号处理、实时滤波、解调
  • 复信号（中心频率 0Hz 的 I/Q 序列）：利用正、负频率区域完整表达基带信息
  • 频谱窗函数（汉明、升余弦、布莱克曼等）：改善频谱显示效果，减小频谱泄漏
  • SDR 数字滤波器类型：FIR、IIR、CIC 等；最灵活，可定制各种通带形状

3.4 电路与元器件（243题）

这是最大的题库章节，涵盖电子电路基础、元器件特性、放大器调谐、仪表使用等。

电子管线性放大器

  • 调谐方法：调整 "Tune" 使屏极电流最小，调整 "Load" 使输出功率最大，然后反复微调
  • 输出扼流圈作用：将输出端直流接地，在屏极耦合电容意外击穿时短路高压并辅助熔断保险丝
  • 机箱联锁开关作用：断开市电输入，降低维修人员误触机内高压的可能性

驻波比（SWR）

  • SWR 趋近于 1:1 最好：降低电缆损耗，使射频能量更有效传输
  • SWR 不稳定可能原因：发信机、馈线或天线某处接触不良
  • 馈线损耗会导致：发信功率降低 和 收信信噪比下降（不会导致 SWR 恒高于 1:1）

同轴电缆

  • 最适合业余无线电的特性阻抗：50 欧姆
  • 室外使用时要求外皮耐紫外线：否则电缆受潮损坏
  • 空气介质同轴电缆的劣势：需特别措施防止湿气渗透
  • M 型同轴电缆连接器：广泛应用于 HF 和 VHF 通信系统
  • 聚乙烯介质同轴电缆速度因子：约 0.65
  • 馈线受潮后果：介质损耗变大；屏蔽层或芯线氧化、锈蚀，甚至断路

天线基础原理

  • 理想点源天线：假想天线，作为比较实际天线辐射性能的全向基准
  • 垂直接地天线（GP）称为"全向天线"：在水平方向上没有指向性
  • 在垂直于振子方向上具有峰值增益的偶极天线：振子长度为 1/2 工作波长（半波偶极子）
  • 水平偶极天线架高降低到距地面 1/10 波长处：馈电点阻抗下降

天线增益：dBi 与 dBd

  • dBi：待测天线最大辐射方向上的辐射功率密度与理想点源天线（各向同性均匀辐射的假想天线）对应值之比的 dB 值
  • dBd：待测天线最大辐射方向上的辐射功率密度与半波长偶极天线（0dBd 基准）对应值之比的 dB 值
  • dBi 与 dBd 换算：dBi = dBd + 2.15（0dBd = 2.15dBi）
  • 说明书只写"dB"不注明基准的增益指标：缺乏参考价值
⚠️ 重点 必记

dBi = dBd + 2.15 — 半波偶极天线本身增益为 2.15dBi（即 0dBd）

增益比较计算

  • 步骤：统一单位（都转为 dBi 或 dBd）→ 差值即为功率比对数
  • 示例：甲天线 6.15dBi，乙天线 1dBd = 2.15dBi → 差值 4dB → 甲信号为乙的 10^(4/10) ≈ 2.5 倍（考试答"两倍"）
  • 示例：甲天线 0dBd = 2.15dBi，乙天线 2dBi → 差值 0.15dB → 甲与半波偶极相当，乙略差
  • 示例：甲天线 4.5dBd = 6.65dBi，乙天线 5.85dBi → 甲比乙强 0.8dB
  • 示例：甲天线 2.9dBd = 5.05dBi，乙天线 5.85dBi → 乙比甲强 0.8dB
  • S 表变化与功率：S表每档 6dB（S1→S9 共 8 档；S9 以上有 +10dB、+20dB 等）
  • 8.15dBi 八木天线换上半波偶极：增益增加 8.15 - 2.15 = 6dB → S 表升 1 档

EIRP 与 ERP

  • EIRP（e.i.r.p.，等效全向辐射功率）：天线输入功率 × 天线增益（以 dBi 为基准的倍数）= P × 10^(GdBi/10)
  • ERP（e.r.p.，有效辐射功率）：天线输入功率 × 天线增益（以 dBd 为基准的倍数）= P × 10^(GdBd/10)
  • 换算：EIRP = ERP × 1.64（即 ERP + 2.15dB）；ERP = EIRP / 1.64
  • 真题示例（3.85dBd → EIRP）：增益 3.85dBd → 转为 dBi = 3.85+2.15 = 6.0dBi → EIRP = 10 × 10^0.6 ≈ 10 × 4 = 40W
  • 真题示例（5.15dBi → ERP）：增益 5.15dBi → 转为 dBd = 5.15-2.15 = 3.0dBd → ERP = 10 × 10^0.3 ≈ 10 × 2 = 20W
⚠️ 重点 考试陷阱

题目中可能出现"等效全向辐射功率 e.i.r.p"但给出的天线增益单位是 dBd。此时需：EIRP = P × 10^((GdBd+2.15)/10)

天线增益测试方法

  • dBd 测试：用同一信号源，调功率使待测天线和参考半波偶极天线在远端产生的场强相同 → 增益(dBd) = 10 log(Po/P)，Po 为送至参考天线的功率，P 为送至待测天线的功率
  • dBi 测试：增益(dBi) = 10 log(Po/P) + 2.15
  • 测试要点：测试距离至少 10 个波长以上（远区场条件）；两天线最大辐射方向对准测试点
  • 场强在远区场：强度与距辐射源的距离的平方成反比

接收机灵敏度

  • 电压灵敏度：最小输入信号电动势（μV 或 dBμV）；数值越小，接收微弱信号能力越强
  • 功率灵敏度：最小输入功率电平（dBm 或 dBμW）
  • 灵敏度换算（50Ω 系统）：
  • 1μV → 2μV（电动势=2×端电压）→ -107dBm → 6dBμV
  • 0.5μV → 1μV（电动势）→ -113dBm → 0dBμV
  • 2μV → -107dBm
  • dBm 计算：P(dBm) = 10 log(V²/R / 1mW)，50Ω 下 1μV 端电压 → -107dBm
  • 比较灵敏度必须知道输出信号质量标准；仅凭电压数值无法直接比较

4.2 传输线与馈线（8题）

同轴电缆特性

  • 业余电台通常选择同轴电缆：易于使用，与环境中其他物体间的互耦低（不是因为损耗最低或最便宜）
  • 特性阻抗：50 欧姆
  • 室外使用需耐紫外线：防止护套破坏后电缆受潮
  • M 型连接器：广泛用于 HF 和 VHF 通信系统
  • 速度因子（聚乙烯介质）：约 0.65
  • 空气介质同轴电缆需防湿气渗透

馈线损耗后果

  • 发信功率降低（主效应）
  • 收信信噪比下降
  • 需要注意：馈线损耗不会导致驻波比的测量值永远高于 1:1

受潮后果

  • 介质损耗变大
  • 屏蔽层或芯线氧化、锈蚀，甚至断路

避雷器安装

  • 将所有避雷器的地线接到同一个金属条带上，然后将条带可靠连到室外的接地处
  • 不要单独引出各避雷器的接地线
  • 不要用氖灯或气体放电管 DIY 避雷器

第五章 安全与规范

5.1 电磁兼容与干扰（5题）

干扰的分类与处理

  • 干扰定义：由于一种或多种发射、辐射、感应或其组合所产生的无用能量对无线电通信系统接收产生的影响（表现为性能下降、误解或信息丢失）
  • 有害干扰：危害无线电导航或其他安全业务正常运行；严重损害、阻碍或一再阻断按规定正常开展的无线电通信业务
  • 遭遇违法/不明电台严重干扰时正确做法：不予理睬，收集信息并向无线电管理机构举报（不要在频率上回应，不要用大功率压制）

防雷接地

  • 防雷接地的作用：把接闪器引入的雷击电流有效地泄入大地
  • 基本要求：单独的接地体，接地电阻要小，引下线应尽量短而粗
  • 不能用自来水管作为接地体，不能用交流电网"零线"代替
  • 引下线不能用细导线（0.5毫米是不够的）

5.3 电源与电池（16题）

直流电源

  • 电源线尽量短而粗：减小发信时电源线上的压降
  • 车载电台直流电源负极：接到电池负极或发动机接地带（不能接在任意金属部件上）

电池管理

  • 电网停电时给 12V 铅酸蓄电池充电：将待充电蓄电池与汽车蓄电池并联，然后发动车辆
  • 电池串联一个电灯泡接到 220V 市电是危险的，不能这样做
  • 判断干电池是否失效：用万用表电压档测量端电压，显著低于标称电压表明失效

电路基本计算（并联/串联）

条件 R1 = N × R2
并联（电压相同） I₁ = I₂/N，P₁ = P₂/N
串联（电流相同） U₁ = N × U₂，P₁ = N × P₂
⚠️ 重点 并联/串联功率计算

- 并联：电压相同，电阻大的电流小 → 功率与电阻成反比（R 越大 P 越小）

- 串联：电流相同，电阻大的电压大 → 功率与电阻成正比（R 越大 P 越大）

N 个相同电阻串并联规律

连接方式 各电阻电压 各电阻电流 各电阻功率 总功率
串联 N 个 减为 1/N 减为 1/N 减为 1/N² = 单个功率 × 1/N
并联 N 个 不变 不变 不变 = 单个功率 × N

欧姆定律与功率公式

  • 已知 U、I：R = U/I，P = U × I
  • 已知 U、R：I = U/R，P = U²/R
  • 已知 I、R：U = I×R，P = I²×R
  • 电压提高 n%：耗电量变为原来 [(100+n)/100]² 倍
  • 电压降低 n%：耗电量变为原来 [(100-n)/100]² 倍

RC / RL 时间常数

  • τ = R × C（RC 串联/并联时间常数）
  • τ = L / R（RL 串联时间常数）
  • RC 串联接直流 U：
  • t=0：电容电压=0，电阻电压=U，电流=U/R
  • t=τ：电容电压升至约 63%U，电阻电压降至约 37%U
  • t→∞：电容电压=U，电阻电压=0，电流=0
  • RC 并联断电源：电容电压从 U 按指数衰减至 0
  • RL 串联接直流 U：
  • t=0：电感电压=U，电阻电压=0，电流=0
  • t=τ：电感电压降至约 37%U，电阻电压升至约 63%U
  • t→∞：电感电压=0，电阻电压=U，电流=U/R

电源类型与效率

  • 线性电源：变压器降压 → 整流 → 滤波 → 稳压。笨重但噪声低
  • 开关电源：整流→滤波→高压DC→开关电路→高频变压器→低压脉冲→整流→滤波。轻巧但谐波辐射会造成电磁干扰
  • 交流市电制 220V 设备：火线中串保险丝；高压整流装置滤波电容需并联泄放电阻
  • 12V 铅酸蓄电池为电台供电风险：通风不良时爆炸性氢气积聚；充放电过快可能过热、释放氢气
  • 电池供电时长估算：标称安时数 / 平均工作电流
  • 电网停电给铅酸蓄电池充电：与汽车蓄电池并联，发动车辆

放大器效率排序

  • 效率由低到高：甲类(A) < 甲乙类(AB) < 乙类(B) < 丙类(C) < 丁类(D)
  • 甲类效率最低（全周期导通，静态功耗大）
  • 丁类/丙类效率最高（开关/窄脉冲工作）
  • 推挽放大电路：实现极小静态偏置电流下的高电源效率线性功率放大

变压器与阻抗变换

  • 匝数比 = 电压比：Vs/Vp = Ns/Np
  • 阻抗比 = (匝数比)²：Zp/Zs = (Np/Ns)²
  • 初级 1100 匝、次级 110 匝接 220V：次级电压 22V；次级 22Ω → 次级电流 1A → 功率 22W
  • 天线阻抗匹配：450Ω 天线 → 50Ω 馈线，需要变压器匝数比 = √(450/50) = 3:1

二极管与整流

  • 硅整流二极管正向压降：约 0.7V
  • 锗二极管：约 0.3V（起始导通电压低 → 简易场强表更灵敏）
  • 50Hz 交流全波整流 → 负载两端为 100Hz 脉动直流
  • 整流滤波电路关机后，滤波电容残留高压——需泄放电阻
  • 防反接保护：电源线串保险丝，反向并联大电流二极管 → 接反时短路烧保险
  • 继电器线圈两端并反向二极管或小电容 → 吸收关断瞬间的反向电动势

其他电源安全

  • 带电检修 220V 设备：双脚与地绝缘，单手操作，另一只手不碰任何金属件
  • 灭火：先切断所有供电，用干式/CO₂/1211灭火器
  • 自制 220V 设备：三芯电源线 + 金属机箱 → 机箱与市电地线可靠连接
  • 不能用自来水管作接地体，不能用交流电网"零线"代替
